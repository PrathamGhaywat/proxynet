# ProxyNet - A reliable and opensource alternative to cloudflare for local curling
ProxyNet is an open-source project that provides a reliable and efficient alternative to cloudflare for simple local curl proxying needs. It is designed to be easy to setup and use. 
Check out the [documentation](USAGE.md) for more details on how to get started. 

ProxyNet is currently in active development and is at the moment closed for contributions. If you have any questions or suggestions, please open an issue.
